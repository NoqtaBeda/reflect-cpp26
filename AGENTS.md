## Introduction

This project, named `reflect_cpp26`, is a comprehensive library with C++26 Reflection (p2996).

This project uses XMake to build & run unit tests.

Local configuration (detailed commands to build & run test cases, for example, where to find `libstdc++.so` or `libc++.so`) can be found in `./AGENTS.local.md` (if exists).

## Project Structure

* `./include/reflect_cpp26`: API & implementation of `reflect_cpp26` library;
* `./docs`: Documentation of components;
* `./examples`: Examples of usage;
* `./tests`: Unit test cases.
